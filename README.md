## Features

- User & Admin roles
- JWT authentication
- Reserve, modify or cancel reservations
- Ecological car suggestions (eco-flagged choices)
- Scheduled email reminders for upcoming reservations
- List & filter available resources
- View reservation history
- Add comments & failure reports
- Admin endpoints for resource management & statistics

## Technologies

- Java 17
- Spring Boot 3.4.3
- Spring Security + JWT
- Spring Mail
- PostgreSQL
- Liquibase
- Swagger
